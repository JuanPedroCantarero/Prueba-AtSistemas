# Concepto Juan Pedro Cantarero Reyes

Concept of Juan Pedro Cantarero with Vue 3 composition api and json_server

## Install proyects

**Back**

```
cd back
npm install
```

**Front**

```
cd front/concept-front
yarn install
```

## How to init

**Back**

```
cd back
npm start
```

**Front**

```
cd front/concept-front
yarn serve
```
